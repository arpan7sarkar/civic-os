# CivicOS Redesign Master Prompt Guide

This document provides a state-of-the-art redesign strategy for CivicOS using **Google Gemini** (Logic/Reasoning), **Google Flow** (High-End Asset Generation), and **Google Stitch** (Component Assembly & Export).

## 0. The Design Vision: "National Command Center"
**Current State**: Functional, government-adherent, reliable.
**Vision**: A premium, "Glassmorphic" national infrastructure portal. Immersive, interactive, and high-fidelity. Think "Apple Design meets Government Transparency."

### Universal Design Tokens (Tailwind 4 & CSS)
- **Base Color**: `gov-blue` (#1e6d8a)
- **Accent**: `primary` (#0EA5A4)
- **Theme**: Light/Dark Hybrid (Glassmorphism)
- **Glass**: `bg-white/10 backdrop-blur-md border hover:bg-white/20 transition-all`
- **Typography**: Inter (Body), Roboto (Headers - Black/Bold weight).

## 1. Redesign Workflow

### Phase 1: Visual Blueprint (Google Flow)
Use **Google Flow** (Model: Nano Banana Pro / Imagen 4) to generate high-fidelity background textures, cinematic 3D icons, and hero visuals.

### Phase 2: Component Architecture (Google Stitch)
Use **Google Stitch** (Model: Gemini 3.0 Flash) to assemble components.
- **Export Mode**: Next.js 16 / Tailwind 4.
- **Constraint**: Ensure all exports use the `createAppwriteClient` pattern for data fetching.

### Phase 3: Logic & Prompt Refinement (Google Gemini)
Use **Gemini 2.0 Pro** to rewrite interactions, AI assistant personalities, and multi-lingual processing logic.

## 3. Implementation Instructions for the Engineer

1.  **Stitch Integration**: When exporting from Stitch, immediately map props to the `AppDataStore` (Zustand) in `src/lib/store.ts`.
2.  **Flow Assets**: Store all assets generated by Flow in `/public/assets/3d/` or `/public/assets/textures/`.
3.  **Tailwind 4 Implementation**: 
    - Use `@theme` for color injections.
    - Avoid `!important`; use the new cascaded layers.
4.  **AI Synergy**: Every redesign must include a `CitizenAIAssistant` component that hooks into the Gemini API in `src/lib/gemini.ts`.

---

## 4. Problem Solving Strategy
- **Redirection Loops**: Ensure `useLayoutEffect` is used for authentication guards to prevent flickering.
- **Data Glitches**: All form inputs from Stitch must include validation masks for National Govt IDs (e.g., Aadhaar 12-digit masking).
- **Scale**: Layouts must support "National Scale" data (millions of records) via virtualized lists in the Grievance History.
